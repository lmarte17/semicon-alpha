# Notebook Templates

These files are plain Python notebook templates for Zerve.

Use them as:

- copy/paste starting points for Zerve notebook cells
- local scripts during development
- a stable sequence for the ingestion workflow

## Order

1. `00_bootstrap.py`
2. `01_news_ingestion.py`
3. `02_reference_sync.py`
4. `03_market_sync.py`
5. `04_workspace_summary.py`

All of them rely on the workflow helpers in `semicon_alpha.workflows`.
